-------------------------------------------------------English------------------------------------------------------

DGMT / Drumgod Modtool is a beta browser extension for Twitch moderation.

The tool provides a local moderation dashboard designed to help Twitch moderators monitor chat activity, review flagged messages, analyze user behavior, save local moderator notes, prepare moderation commands, and manage stream-related tools from one place.

Current features include live chat data display, filtered message detection, local message analysis, user analysis, local moderator notes, selected-message tracking, AutoMod-style queue handling, chat log recording/export, action logging, timeout/warn/ban command preparation, shoutout command preparation, raid command tools, stream activity command helpers, collapsible panels, language switching, custom appearance settings, custom color gradients, custom backgrounds, and a draggable Twitch overlay button.

Stream activity tools currently include raid command preparation, flipcoin, 8ball, dice, giveaway/raffle, queue, quote, and songrequest command helpers.

At the current beta stage, moderation and stream activity commands are prepared locally and can be copied for manual use. Any feature that sends commands directly to Twitch chat or performs moderation actions through Twitch is marked as WIP and requires Twitch API access.

Planned API features include Twitch OAuth login, real moderator-right verification, direct chat command sending, direct timeout/ban/delete/pin actions, AutoMod approval/denial, direct shoutouts, ad controls, and editable stream information.

This version is intended for beta testing and feedback from active Twitch moderators.

For the full installation guide, user guide, and beta feedback, please join my Discord:
https://discord.gg/TeW5EDyStr

--------------------------------------------------------Deutsch-----------------------------------------------------

DGMT / Drumgod Modtool ist eine Beta-Browser-Erweiterung für Twitch-Moderation.

Das Tool stellt ein lokales Moderations-Dashboard bereit, das Twitch-Moderatoren dabei helfen soll, Chataktivität zu überwachen, auffällige Nachrichten zu prüfen, User-Verhalten zu analysieren, lokale Moderator-Notizen zu speichern, Moderationsbefehle vorzubereiten und streambezogene Tools übersichtlich an einem Ort zu verwalten.

Aktuelle Funktionen umfassen Live-Chatdatenanzeige, Erkennung gefilterter Nachrichten, lokale Nachrichtenanalyse, User-Analyse, lokale Moderator-Notizen, ausgewählte Nachrichtenverfolgung, AutoMod-ähnliche Queue-Verwaltung, Chatlog-Aufzeichnung/Export, Action Log, Timeout-/Warn-/Ban-Befehlserstellung, Shoutout-Befehlserstellung, Raid-Command-Tools, Stream-Aktivitäts-Command-Helfer, einklappbare Panels, Sprachumschaltung, eigene Darstellungseinstellungen, eigene Farbverläufe, eigene Hintergründe und ein verschiebbares Twitch-Overlay-Logo.

Die Stream-Aktivitätstools umfassen aktuell vorbereitete Raid-Befehle, Flipcoin, 8ball, Dice, Giveaway/Raffle, Queue, Quote und Songrequest Command-Helfer.

Im aktuellen Beta-Stand werden Moderations- und Stream-Aktivitätsbefehle lokal vorbereitet und können zur manuellen Nutzung kopiert werden. Jede Funktion, die Befehle direkt in den Twitch-Chat sendet oder Moderationsaktionen direkt über Twitch ausführt, ist als WIP markiert und benötigt Twitch-API-Zugriff.

Geplante API-Funktionen umfassen Twitch OAuth Login, Modrechte-Prüfung, direktes Senden von Chatbefehlen, direkte Timeout-/Ban-/Delete-/Pin-Aktionen, AutoMod-Freigabe/Ablehnung, direkte Shoutouts, Werbesteuerung und bearbeitbare Streaminfos.

Diese Version ist für Beta-Tests und Feedback von aktiven Twitch-Moderatoren gedacht.
