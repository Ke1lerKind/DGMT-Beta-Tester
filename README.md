-------------------------------------------------------English------------------------------------------------------

DGMT / Drumgod Modtool is a beta browser extension for Twitch moderation.

The tool provides a local moderation dashboard designed to help Twitch moderators monitor chat activity, review flagged messages, analyze user behavior, save local moderator notes, prepare moderation commands, and manage stream-related tools from one place.

Current features include live chat data display, filtered message detection, local message analysis, user analysis, local moderator notes, selected-message tracking, AutoMod-style queue handling, chat log recording/export, action logging, timeout/warn/ban command preparation, shoutout command preparation, raid command tools, stream activity command tools, collapsible panels, language switching, custom appearance settings, custom color gradients, custom backgrounds, and a draggable Twitch overlay button.

Stream activity tools currently include raid commands, flipcoin, 8ball, dice, giveaway/raffle, queue, quote, and songrequest command helpers.

Some commands can be copied or sent into Twitch chat directly through the browser extension. Features that require direct Twitch API access are still marked as WIP. Planned API features include Twitch OAuth login, real moderator-right verification, direct timeout/ban/delete/pin actions, AutoMod approval/denial, direct shoutouts, ad controls, and editable stream information.

This version is intended for beta testing and feedback from active Twitch moderators.

For the full installation guide, user guide, and beta feedback, please join my Discord:
https://discord.gg/TeW5EDyStr

--------------------------------------------------------Deutsch-----------------------------------------------------

DGMT / Drumgod Modtool ist eine Beta-Browser-Erweiterung für Twitch-Moderation.

Das Tool stellt ein lokales Moderations-Dashboard bereit, das Twitch-Moderatoren dabei helfen soll, Chataktivität zu überwachen, auffällige Nachrichten zu prüfen, User-Verhalten zu analysieren, lokale Moderator-Notizen zu speichern, Moderationsbefehle vorzubereiten und streambezogene Tools übersichtlich an einem Ort zu verwalten.

Aktuelle Funktionen umfassen Live-Chatdatenanzeige, Erkennung gefilterter Nachrichten, lokale Nachrichtenanalyse, User-Analyse, lokale Moderator-Notizen, ausgewählte Nachrichtenverfolgung, AutoMod-ähnliche Queue-Verwaltung, Chatlog-Aufzeichnung/Export, Action Log, Timeout-/Warn-/Ban-Befehlserstellung, Shoutout-Befehlserstellung, Raid-Command-Tools, Stream-Aktivitätsbefehle, einklappbare Panels, Sprachumschaltung, eigene Darstellungseinstellungen, eigene Farbverläufe, eigene Hintergründe und ein verschiebbares Twitch-Overlay-Logo.

Die Stream-Aktivitätstools umfassen aktuell Raid-Befehle, Flipcoin, 8ball, Dice, Giveaway/Raffle, Queue, Quote und Songrequest Command-Helfer.

Einige Befehle können kopiert oder direkt über die Browser-Erweiterung in den Twitch-Chat gesendet werden. Funktionen, die direkten Twitch-API-Zugriff benötigen, sind weiterhin als WIP markiert. Geplante API-Funktionen umfassen Twitch OAuth Login, echte Modrechte-Prüfung, direkte Timeout-/Ban-/Delete-/Pin-Aktionen, AutoMod-Freigabe/Ablehnung, direkte Shoutouts, Werbesteuerung und bearbeitbare Streaminfos.

Diese Version ist für Beta-Tests und Feedback von aktiven Twitch-Moderatoren gedacht.
